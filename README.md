# Social

## Get Satrted

- Clone repo
- - run in terminal `git clone https://github.com/juxsalley/social.git`
- - cd social and install gems by running `bundle install`
- - and also install yarn packages using `yarn install `

- create database
- - `rails db:create` and `rails db:migrate` in terminal

- run
- - `rails s` in terminal to start server

- run test
- - `bundle exec rspec` in terminal
